Getting Started
1. First, install pandas and openpyxl:

pip install pandas openpyxl


2. Let’s assume we have an Excel file called “grades.xlsx” with the following data:

Name Math English Science
Alice 85 90 88
Bob 78 82 84
Charlie 92 87 89

1. Reading the Excel File

This code opens the file and shows the data.

import pandas as pd

# Read the Excel file
data = pd.read_excel("grades.xlsx")

# Show the data
print(data)

Output:

      Name  Math  English  Science
0    Alice    85       90       88
1      Bob    78       82       84
2  Charlie    92       87       89

2. Add a New Column

We’ll calculate the total score for each student and add it as a new column.

# Add a "Total" column
data["Total"] = data["Math"] + data["English"] + data["Science"]

print(data)

Output:

      Name  Math  English  Science  Total
0    Alice    85       90       88    263
1      Bob    78       82       84    244
2  Charlie    92       87       89    268

3. Update a Specific Value

Suppose Bob’s Math grade was entered incorrectly. Let’s change it to 80.

# Update Bob's Math grade
data.loc[data["Name"] == "Bob", "Math"] = 80

print(data)

Output:

      Name  Math  English  Science  Total
0    Alice    85       90       88    263
1      Bob    80       82       84    246
2  Charlie    92       87       89    268

4. Filter Data

Let’s find students who scored more than 250 in total.

# Filter students with Total > 250
high_scorers = data[data["Total"] > 250]

print(high_scorers)

Output:

      Name  Math  English  Science  Total
0    Alice    85       90       88    263
2  Charlie    92       87       89    268

5. Save the Modified File

Finally, we save the updated data to a new Excel file.

# Save the updated data to a new file
data.to_excel("updated_grades.xlsx", index=False)

Summary
• Read: Open and view the data from the Excel file.
• Manipulate: Add columns, update values, or filter rows.
• Save: Save the modified data to a new Excel file.
